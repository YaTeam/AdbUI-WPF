# ADB Studio

Открытое (open-source), современное и лёгкое настольное приложение для Windows, предназначенное для управления Android-устройствами через **ADB** и **scrcpy** — без необходимости работать с командной строкой.

![Platform](https://img.shields.io/badge/platform-Windows-blue)
![.NET](https://img.shields.io/badge/.NET-WPF-512BD4)
![Open Source](https://img.shields.io/badge/open%20source-%E2%9D%A4-red)

Это бесплатный проект с открытым исходным кодом — вы можете изучать код, использовать его, изменять и вносить свой вклад.

## ✨ Возможности

- 📱 **Управление устройствами** — автоматическое определение подключённых устройств, отображение заряда батареи, памяти, ОЗУ и версии Android
- 🖥️ **Трансляция экрана** — запуск scrcpy прямо из приложения в один клик
- 🖲️ **Аппаратные кнопки** — мгновенная отправка событий Home, Back, Menu, увеличения/уменьшения громкости
- 📋 **Просмотр Logcat в реальном времени** — фильтрация, поиск и пауза потока логов
- ⚙️ **Монитор процессов** — загрузка CPU/RAM по каждому процессу в реальном времени с автообновлением
- 🔌 **Меню питания** — перезагрузка в систему, Recovery, Bootloader, а также безопасное выключение
- 🧰 **Быстрые и пользовательские команды (Hot Commands)** — сохраняйте свои часто используемые ADB-команды как кнопки в один клик
- 🗂️ **Свои пути к инструментам** — укажите собственные папки с ADB / scrcpy или используйте системный PATH
- 🎨 **Чистый современный интерфейс** — собственный дизайн на WPF в стиле Material

## 🚀 Начало работы

### Требования
- Windows 10/11
- [8.0 NET Desktop Runtime]((https://dotnet.microsoft.com/download/dotnet/8.0))
- [ADB (Android Platform Tools)](https://developer.android.com/tools/releases/platform-tools)
- [scrcpy](https://github.com/Genymobile/sпоcrcpy) (опционально, требуется для трансляции экрана)

### Установка
1. Скачайте последнюю версию со страницы [Releases](../../releases)
2. Распакуйте архив и запустите `ADBStudio.exe`
3. При первом запуске поместите `adb.exe`/`scrcpy.exe` рядом с приложением, добавьте их в системный PATH, либо укажите свои папки в **Настройках**

### Использование
1. Подключите Android-устройство по USB с включённой отладкой по USB
2. Устройство автоматически появится в выпадающем списке сверху
3. Используйте боковую панель для трансляции экрана, отправки аппаратных клавиш или управления питанием
4. Переключайтесь между вкладками **Devices**, **Logcat** и **Processes** через левую панель навигации

## ⚙️ Настройки

- Включение собственной директории для ADB/scrcpy (вместо папки приложения или системного PATH)
- Настройка интервала автообновления списка устройств
- Включение/отключение подтверждения перед перезагрузкой/выключением
- Добавление собственных **Hot Commands** — пользовательских ADB shell-команд, которые появляются как кнопки быстрого доступа
